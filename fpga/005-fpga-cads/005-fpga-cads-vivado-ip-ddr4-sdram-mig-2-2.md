# DDR4 SDRAM (MIG) (2.2)

### Комментарии к конфигурации IP модуля

- Параметр *Memory Device Interface Speed* прямо связан с частотой тактирования модуля и AXI.
- Для параметра *Data Mask and DBI* необходимо выбрать значение **DM DBI RD**

### Комментарии к подключению модели DDR4 в tb

- Сгенерировать IP блок
- Добавить его в проект
- Создать тестовый модуль ddr4 c портами, соответствующими физическим.
- Создать в *tb_top.sv* экземпляр р модуля ddr4.
- Создать проект Example для DDR4 SDRAM (MIG)
- Открыть
