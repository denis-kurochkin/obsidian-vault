## Введение

Источник: https://adaptivesupport.amd.com/s/article/1053914?language=en_US

**AXI4** — это высокопроизводительный интерфейс системной шины из стандарта **AMBA 4**, разработанный компанией **Arm**. Он широко используется в **SoC, FPGA (Vivado), процессорах, DMA, контроллерах памяти**.

### Основные положения AXI4

AXI4 основан на **разделении адреса, данных и управления** и использует **независимые каналы**, что позволяет:

- передавать данные параллельно,
- увеличивать пропускную способность,
- уменьшать задержки.

**Master** инициирует транзакции  
**Slave** отвечает на них

AXI4 имеет 5 независимых каналов AXI4

2 используются для операций чтения
 - чтение адреса
- чтение данных
3 используются для операций записи
- запись адреса
- запись данных
- запись ответа

![[Pasted image 20260123140134.png]]

Для **каждого канала** есть сигналы:

- `VALID` — источник говорит: _«данные готовы»_
- `READY` — приёмник говорит: _«готов принять»_

📌 Передача происходит **только когда `VALID = 1` и `READY = 1` одновременно**  
📌 Это позволяет легко делать back-pressure

![[Pasted image 20260123140557.png]]

### Чтение (Read Transaction)

1. Master передаёт адрес → **AR**
2. Slave возвращает данные → **R**

![[Pasted image 20260123140850.png]]

### Запись (Write Transaction)

1. Master выставляет адрес → **AW**
2. Master передаёт данные → **W**
3. Slave подтверждает → **B**

![[Pasted image 20260123141055.png]]

Важно:
- адрес и данные могут приходить **не одновременно**
- поддерживаются **burst-транзакции**

**Burst-передачи**
AXI4 поддерживает burst-доступы (очень важно для DDR):

| Тип   | Описание            |
| ----- | ------------------- |
| FIXED | адрес не меняется   |
| INCR  | адрес увеличивается |
| WRAP  | адрес зацикливается |

**Ответы (RESP)**

Коды ответа (`BRESP`, `RRESP`):

| Код      | Описание                    | Значение |
| -------- | --------------------------- | -------- |
| `OKAY`   | успешно                     | 0b00     |
| `EXOKAY` | эксклюзивный доступ         | 0b01     |
| `SLVERR` | ошибка slave                | 0b10     |
| `DECERR` | ошибка декодирования адреса | 0b11     |

### Варианты AXI4

- AXI4             - Стандартный вариант. Используется для доступа к памяти и т.д.
- AXI4-Stream   
- AXI4-Lite

## Описание работы

### Описание сигналов:

**Канал адреса**

| WA channel | RA channel | Описание                                                                                                                                                                                                                                                                         |
| ---------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| AWID       | ARID       | ID устройства, для адресации внутри одного канала.                                                                                                                                                                                                                               |
| AWADDR     | ARADDR     | Адрес первого ***beat*** в ***burst***                                                                                                                                                                                                                                           |
| AWLEN      | ARLEN      | Количество ***beat*** в ***burst***                                                                                                                                                                                                                                              |
| AWSIZE     | ARSIZE     | Размер одного ***beat***<br><br>0 - 1 byte<br>1 - 2 bytes<br>2 - 4 bytes<br>3 - 8 bytes<br>. . .<br>7 - 128 bytes                                                                                                                                                                |
| AWBURST    | ARBURST    | Тип ***burst***<br><br>0b00 - FIXED<br>0b01 - INCR<br>0b10 - WRAP                                                                                                                                                                                                                |
| AWLOCK     | ARLOCK     | Тип блокировки, обеспечивающий атомарные операции.                                                                                                                                                                                                                               |
| AWCACHE    | ARCACHE    | Тип памяти, способ продвижения транзакции через систему.                                                                                                                                                                                                                         |
| AWPROT     | ARPROT     | Тип защиты: привилегии, уровень безопасности и доступ к данным/инструкциям.<br><br>0bxx0 - Непривилегированный доступ<br>0bxx1 - Привилегированный доступ<br>0bx0x - Защищенный доступ<br>0bx1x - Незащищенный доступ<br>0b0xx - Доступ к данным<br>0b1xx - Доступ к инструкциям |
| AWQOS      | ARQOS      | Качество обслуживания передачи                                                                                                                                                                                                                                                   |
| AWREGION   | ARREGION   | Идентификатор региона, позволяющий получить доступ к нескольким логическим интерфейсам с одного физического интерфейса.                                                                                                                                                          |
| AWUSER     | ARUSER     | Данные, определяемые пользователем                                                                                                                                                                                                                                               |
| AWVALID    | ARVALID    | Сигнал handshake                                                                                                                                                                                                                                                                 |
| AWREADY    | ARREADY    | Сигнал handshake                                                                                                                                                                                                                                                                 |
